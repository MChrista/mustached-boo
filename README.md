# Verteilte Systeme
<h2>Description</h2>
A little server returning the time in rfc 2616 standard to incoming connections.
<h2>Steps to compile:</h2>
<ul>
	<li>run "make libs"</li>
	<li>run "make"</li>
</ul>
<h2>Steps to clean:</h2>
<ul>
	<li>run "make clean libs"</li>
	<li>run "make clean"</li>
</ul>
